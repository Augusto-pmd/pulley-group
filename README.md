# Pulley Group

Sistema de gestión patrimonial con diseño Apple macOS.

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Estructura

- `/app` - Páginas y layout de Next.js
- `/components` - Componentes reutilizables
- `/mock` - Datos mock para desarrollo
- `/styles` - Estilos globales
- `/design` - Documentación de diseño visual
- `/ux` - Documentación de UX por pantalla

## Notas

- No hay backend ni base de datos
- Todos los datos son mock
- El diseño respeta estrictamente el lenguaje visual definido en `/design/visual-language.md`
- El UX respeta estrictamente las especificaciones en `/ux/*.md`
